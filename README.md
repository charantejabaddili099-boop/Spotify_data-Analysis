This project analyzes Spotify music data using Python to uncover meaningful insights through data preprocessing, exploratory data analysis (EDA), and visualization.

Features:

Cleaned and preprocessed Spotify dataset.
Handled missing values and removed unnecessary columns.
Converted song duration from minutes to seconds.
Identified top artists based on followers and popularity.
Analyzed the relationship between song duration and popularity.
Created a correlation heatmap to identify factors affecting track popularity.
Generated high-quality visualizations using Matplotlib and Seaborn.

Technologies Used:

Python
Pandas
NumPy
Matplotlib
Seaborn

Key Insights:

Artists with larger follower bases generally achieve higher popularity.
Most popular songs are around 3–4 minutes long, although duration alone has little impact on popularity.
Artist popularity and follower count have the strongest positive correlation among the analyzed features.
